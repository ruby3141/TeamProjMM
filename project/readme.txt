큰 틀에서 방과 그 방안에 그룹이 있음

처음에 로그인은 되었다고 가정 (db에 dummy값 존재)

홈페이지에 들어온후 마스터는 방을 만듦 (중요한것은 마감일이 있는것) 그 후 db에 저장
그 후 find room을 누르면 현재 생성된 방들의 리스트를 보여줌
각 방의 마스터는 자신의 방 삭제 가능 

방들의 마감기한이 지난 것은 enter를 누른 후에 make 및 join이 불가능하고 오직 방안에 그룹만 볼 수 있음.
마감기한이 지나지 않은 방들에 대해서 join을 누른 사람들은 각 방의 db에 연결
이미 조를 만들 사람들이 모여 있으면 make로 그룹을 만들 수 있음. 그후 db 에서 그룹내에 들어 갔다고 update 해줌
마감 기한이 지난 후에 그룹이 없는 인원은 그룹을 자동으로 생성

각 그룹을 클릭하면 그룹원들의 전화번호와 이메일 이름을 알수 있음.
